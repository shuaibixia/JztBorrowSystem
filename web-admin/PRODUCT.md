# 摄影器材管理后台

## Product

面向学生组织器材管理员的桌面运营后台。管理员在这里查看全量数据、处理审批和预约、维护器材与成员信息；小程序仍服务于成员借用、扫码和日常自助操作。

## Users

- `admin`：日常器材、成员、审批、预约、维保和借还操作。
- `superadmin`：在管理员能力上，增加角色调整、成员停用、审计和系统诊断。

## Platform

web

## Register

product

## Success

- 桌面管理员能在一个稳定工作台中找到待处理事项与全量数据。
- 所有写操作使用受控领域规则，不允许直接篡改借还、审批或预约状态。
- 未登录、无权限、会话过期、加载、空数据和失败状态都有清晰路径。

## Constraints

- 使用 React 18、Vite 4、JavaScript 和 Fluent UI React v9。
- 兼容 Node 16.14 以上的本地构建。
- 正式登录使用微信开放平台网站扫码登录；OAuth 上线前仅本地开发可使用模拟身份。
- 浏览器不直接访问 CloudBase NoSQL，也不持有微信 OAuth secret。
